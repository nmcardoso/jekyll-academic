## Code Snippet

```python
from dataclasses import dataclass

@dataclass
class Metric:
    name: str
    value: float


def summarize(metrics: list[Metric]) -> str:
    lines = [f"{m.name}: {m.value:.2f}" for m in metrics]
    return "\n".join(lines)


if __name__ == "__main__":
    sample = [Metric("accuracy", 0.97), Metric("f1", 0.95)]
    print(summarize(sample))
```
